# Story 005：自由瀏覽

## User Story
**As a** 業務 / 任何資料使用者
**I want to** 用多維度 filter 直接翻原始資料（類型/關鍵字/日期/機關類型/只看自家）
**So that** 當儀表板沒有剛好我要的切法時，能自己組合條件找案

## 成功指標
任何 ad-hoc 查詢需求，不用改 code 就能透過 UI 過濾完成。
